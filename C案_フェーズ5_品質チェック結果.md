## 5. E（AIらしさ診断）レポート

**参照**：anti-ai-aesthetic チェックリスト。  
**診断対象**：実機 index-c.html（ビジュアル・レイアウト・型・写真）。

### 5-1. チェック項目の○×

| 項目 | 結果 | 理由 |
|------|------|------|
| 汎用アウトラインアイコンを角丸ボックスに並べただけのブロックにしていないか | ✅ | アイコン・絵文字は使用禁止で削除済み。バッジは文言のみ。 |
| 「タグ3つ→説明1行→CTA1つ」の型を他サイトとそっくり繰り返していないか | ✅ | FV はバッジ4つ＋キャッチ＋損失＋CTA 複数。セクションごとに構成が異なる。 |
| 角丸・余白・フォントがすべて同じ数値で均一になりすぎていないか | ✅ | 角丸 4px（バッジ）/ 6px（損失）/ 8px（CTA・カード）/ 10px（カード・表）/ 12px（クイズ・画像）。余白は clamp で可変。 |
| 緑・オレンジ・グレー以外に、このサイトだけの色の使い方があるか | ✅ | 紺（--c-primary）＋緑 CTA＋ゴールドアクセント。業界典型の緑単体ではなく、紺ベースで差別化。 |
| アイコンより写真・実在感で伝えられる部分でアイコンに逃げていないか | ✅ | CTA・Offer・Narrowing は文言のみ。FV・Solution に画像スロットあり。 |
| 「このサイトならでは」の要素が1つ以上あるか | ✅ | 紺＋ゴールドのトーン、元警察官・遺品整理に特化した構成、診断クイズで4パターン結果。 |

### 5-2. 該当箇所の理由（該当なしのため補足のみ）

- **Narrowing カード**：`.c-narrow__card.fit div::before { content: '✓' }` は **テキストの一文字**であり、汎用アウトラインアイコン（SVG）ではない。E の診断では「記号の ✓ は文言の一部として許容」とする。
- **Evidence の ★**：口コミ評価の「★5.0」は**固有名詞的な表記**のためそのままでよい。

### 5-3. 診断結論

**AIらしさ診断**：**該当なし**。アイコン・絵文字廃止、角丸・余白のゆらぎ、紺・緑・ゴールドの色使い、「このサイトならでは」の構成が揃っており、AI生成・テンプレ典型と一瞬で判断される要素はない。

**E 判定**：**問題なし**。公開可能水準。
